# AI Email Sender

An AI-powered tool that writes and sends personalized emails automatically.

## What it does
- Takes a topic, your name and friend's name as input
- Uses Google Gemini AI to write a professional email
- Shows a preview before sending
- Sends the email via Gmail automatically

## Tech Used
- Python
- Google Gemini API
- smtplib
- python-dotenv

## How to Run
1. Clone the repo
2. Create a .env file and add your GEMINI API key and Gmail App Password
3. Install dependencies: pip install google-genai python-dotenv
4. Run: python main.py
